# mosnaps frontend

MoSnaps takes photos when motion is detected.

## Goals v0

Can be used as security camera with old hardware. Runs in browser with no setup.

- runs locally in a web browser
- can run on a browser that's 10 years old
- no external dependencies
- source code is readable when using view source in a browser
- code should be as simple as possible

## Run locally

Run the following then open a browser to http://localhost:8000 to view the site.

```bash
python3 -m http.server 8000
```
